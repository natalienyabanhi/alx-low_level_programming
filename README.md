## ALX Low-Level C Programming

This repository contains all the C programming projects completed as part of the ALX Software Engineering program in partnership with Holberton School. Each directory represents a specific topic, focusing on mastering data structures, algorithms and system-level programming.

## 📑 Table of Contents

| Directory | Project Description |
|---|---|
| 0x00-hello_world | Introduction to C compilation and basic output |
| 0x01-variables_if_else_while | Conditionals, loops, and basic variable types |
| 0x02-functions_nested_loops | Using functions and nested logic |
| 0x04-more_functions_nested_loops | Further function practice |
| 0x05-pointers_arrays_strings | Introduction to pointers and memory |
| 0x11-printf | Custom implementation of the printf function |
| 0x16-simple_shell | A basic Unix command-line interpreter project |

##  Technologies & Tools

* Language: C (Standard: gnu89)
* Compiler: gcc with flags -Wall -Werror -Wextra -pedantic
* Style: All code follows the Betty style checker
* Debugger: GDB and Valgrind for memory leak detection
* Editor: vi, Vim, or Emacs

##  Getting Started
To test any of the files in this repository, clone the repo and compile using the standard flags:

# Clone the repository
git clone https://github.com
# Enter a directory
cd 0x00-hello_world
# Compile a specific file
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 4-puts.c -o puts
# Execute
./puts
